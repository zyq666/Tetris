# tetris

## A tetris game for Flutter.

* [x] support for iOS

* [x] support for Android


## clone it and run 

 <code>flutter build apk</code>
 
 ### and you will find the apk in path of
 
 `build/app/outputs/apk/release/app-release.apk`
 
 ### then install it in your Android phone.
 
 # now you can run it!
